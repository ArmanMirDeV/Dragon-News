## Description
**Dragon News** is a modern news portal web application where users can browse, read, and explore news from different categories. The platform is fully responsive, with smooth UI interactions, and provides a fast, user-friendly reading experience.

---

## Key Features
- **Category Browsing:** Users can explore news by different categories.  
- **Dynamic Routing:** Single-page application routing with React Router.  
- **Responsive Design:** Optimized layouts for mobile and desktop devices.  
- **Interactive UI:** Smooth animations powered by Framer Motion.  
- **Optional Authentication:** Can be extended with Firebase login/registration.  
- **API Integration:** Fetches news data dynamically using Axios.  
- **Charts & Analytics (Optional):** Recharts integration for visual insights.  
- **Notifications & Alerts:** React Hot Toast and SweetAlert2 integration for feedback.  
- **Modern UI:** TailwindCSS and DaisyUI for clean, fast UI development.  

---

## Tech Stack
- **Frontend:** React, TailwindCSS, DaisyUI, Framer Motion, Lucide React, React Router  
- **Backend (Optional):** Node.js, Express.js, REST APIs  
- **Data Fetching:** Axios  
- **Notifications:** React Hot Toast, SweetAlert2  
- **Build Tools & Dev:** Vite, ESLint
